google-authenticator-box
========================

[Google Authenticator](https://code.google.com/p/google-authenticator/) verification environment box virtual-machine image.

Usage
-----

### Setup git submodules.

```
$ make
```

### Build box.

```
$ make test
```

### Build architecture-specific box.

```
$ make i686
```

```
$ make x86_64
```
